# Natural Health World - Billing & Inventory System

A complete desktop application for managing billing and inventory for Ayurvedic medicine business.

## Features
- Bill generation with discount management
- Inventory management with batch tracking
- Sales reports and analytics
- Customer management
- Database backup and restore
- PDF invoice generation

## Tech Stack
- Python 3.x
- PyQt5 (GUI)
- SQLite (Database)
- ReportLab (PDF generation)

## Installation

1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python main.py`
